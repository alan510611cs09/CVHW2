# Introduction
To perform object detection using DETR architecture, RESNET 50 as backbone.


# Environment Setup
The default running environment is **Google Colab**. Using **G4** GPU.

If running this on Google Colab, all packages should be installed automatically.

Otherwise, use the command to install packages needed:

`
pip install -r requirements.txt
`
